Font Minifier

A small python script that creates 'minified' font files in various web-ready formats. It does this by utilising the python FontForge API.

Very simple little tool that with takes all the glyphs in a font, extracts the ones you want and creates a new font with them.

very useful for those projects that have a very small file-size limit.
