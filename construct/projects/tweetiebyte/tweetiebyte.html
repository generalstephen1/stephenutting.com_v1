TweetieByte

Tweetiebyte is a Twitter scanning infographic generator from pre Twitter analytics days (circa 2014) you can put in any Twitter handle and it will return some handy data in the form of a cute infographic

Delving into the world of HTML5 and Twitter related user behavior, a deceptively simple brief was set. The challenge was to see what useful information we could glean from users Twitter accounts WITHOUT user authentication. The output is TweetieByte  a super simple social media monitoring tool that spits out user-friendly visual data in easy-to-swallow byte size chunks!
